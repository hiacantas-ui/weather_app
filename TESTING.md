# 테스트 체크리스트

## 기능 테스트

### 1. 도시 선택 기능
- [ ] 드롭다운에서 도시 선택 시 날씨 데이터가 변경됨
- [ ] 상단 Header의 "현재 지역"도 함께 업데이트됨
- [ ] 10개 모든 도시 (서울, 부산, 인천, 대구, 대전, 광주, 울산, 수원, 세종, 제주) 테스트
- [ ] 페이지 새로고침 후에도 마지막 선택 도시가 유지됨

### 2. 현재 날씨 표시
- [ ] 온도가 실시간으로 표시됨
- [ ] 날씨 아이콘이 정상 표시됨
- [ ] 체감온도, 습도, 풍속, 가시거리 모두 표시됨
- [ ] 날씨 상태에 따라 배경 그라데이션이 변경됨
- [ ] 온도 조언 메시지가 표시됨
- [ ] 날씨 이모지가 표시됨

### 3. 예보 정보
- [ ] "시간별" 탭 클릭 시 시간별 예보 8개 표시
- [ ] "일별" 탭 클릭 시 일별 예보 5일 표시
- [ ] 각 예보에 날씨 아이콘, 온도, 강수확률 표시
- [ ] 탭 전환이 부드럽게 동작함

### 4. 로딩 상태
- [ ] 도시 변경 시 스켈레톤 UI 표시
- [ ] 로딩 완료 후 페이드인 애니메이션 적용
- [ ] 스켈레톤 UI가 실제 카드 구조와 유사함

### 5. 에러 처리
- [ ] API 오류 시 에러 메시지 표시
- [ ] "다시 시도" 버튼이 동작함
- [ ] 에러 메시지가 명확함

## 반응형 테스트

### 모바일 (< 640px)
- [ ] Header 아이콘과 텍스트가 작게 표시됨
- [ ] "현재 지역:" 텍스트가 숨겨짐
- [ ] 현재 날씨 카드가 세로로 배치됨
- [ ] 시간별 예보가 2열 그리드로 표시됨
- [ ] 모든 텍스트가 읽기 쉬움

### 태블릿 (640px ~ 768px)
- [ ] 중간 크기로 잘 표시됨
- [ ] 레이아웃이 자연스러움

### 데스크톱 (> 768px)
- [ ] 현재 날씨 카드가 가로로 배치됨
- [ ] 시간별 예보가 4열 그리드로 표시됨
- [ ] 전체 레이아웃이 중앙 정렬됨

### 브라우저 창 크기 변경
- [ ] 창 크기를 동적으로 변경해도 레이아웃이 깨지지 않음
